definicion apache2ctl
interface de control apache que sirve para ayudar al administrador a controlar el funcionamiento del demonio APACHE apache2 
parametros:
 - start (inicia el demonio apache)
 - stop (para el demonio apache)
 - status (muestra un breve informe de estado. Es parecido a la opcion fullstatus excepto que se omite la lista de peticiones que se estan sirviendo actualmente).
 - fullstatus (muestra un informe de estado completo usando mood_status para ello necesita tener mood_status habilitado en nuestro servidor y un navegador en modo texto como lynx disponible en el sistema)
 - restart (reinicia del demonio apache enviandole la se√±al SIGHUP si el demonio no esta en ejecucion se inicia. este comando checkea automaticamente los ficheros de configuracion via configtest antes de comenzar el reinicio para cazar los errores mas obvios)
 - configtest (ejecuta un test de sintasis de fichero de fichero de configuracion)
 - help
